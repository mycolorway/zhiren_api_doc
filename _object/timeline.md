Timeline
{: .success }

属性            | 类型  | 默认值 | 描述         |
----------------|---------|--------|----------------------|
 date           | 日期    |        | 发生日期             |
 category       | 字符串  |        | leave: 请假; business_trip: 公出 |
 duration       | 数字    |        | 时间         |
 duration_unit  | 字符串 |        |  days: 天；hours: 小时；minutes: 分钟 |
 start_time     | 时间    |        | 开始时间      |
 end_time       | 时间    |        | 结束时间      |
