Approval
{: .info }

属性           | 类型  | 默认值 | 描述         |
---------------|-------|--------|--------------|
number_series  |  字符串 |       | 审批单号     |
creator        |  字符串 |       | 发起人       |
created_at     |  时间   |       | 发起时间     |
status         |  字符串 |       | 审批状态     |
status_name    |  字符串 |       | 状态名称     |
updated_at     |  字符串 |       | 最后审批时间  |
category       |  字符串 |       | 审批类型     |
workflow_name  |  字符串 |       | 审批流名称   |
form_values    | 对象   |        | 审批表单内容，和审批类型有关 |
